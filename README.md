# 🤖 AI Chatbot using Hugging Face Transformers

## 📌 Project Overview
This project demonstrates a simple AI chatbot built using a pre-trained language model from Hugging Face Transformers.  
It uses prompt-based interaction to generate text responses for user questions.

---

## 🎯 Objective
To understand and implement:
- LLM pipeline integration
- Prompt engineering basics
- Text generation using transformer models

---

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook

---

## 🤖 Model Used
- distilgpt2 (lightweight GPT-2 variant)

---

## ⚙️ How It Works
1. User inputs a question
2. Input is passed to a pre-trained language model pipeline
3. Model generates a text-based response
4. Output is displayed in notebook

---

## ⚠️ Limitations
- Model is not instruction-tuned
- Responses may be repetitive or generic
- Designed for learning purposes only

---

## 📌 Key Learning
- How LLM pipelines work
- Basic prompt engineering
- Model integration in Python notebooks
